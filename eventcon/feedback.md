Personal feedback:

- part of the homepage has a weird color inconsistency ([003](https://www.loom.com/share/494bf51e94fe47b1a8842f8e8a73a961))
- looking at the perks of a meetup organizer it looks like I would be ale to
  have unlimited attendees but looking at the stats for an event rsvp I can only
  get 50 rsvps? It would be cool to be able to change this max rsvp number so
  depending on the size of the venue holding the event it can be represented by
  a max rsvp value
- I felt the search for events a little lacking, specifically the filtering I
  expected to be able to filter through date ranges or type of meeting (virtual,
  in person, conference)
- also think that once there are more conferences and events in general the
  "upcoming events" section of  the event search could get a little crowded and
  redundant. I would have expected to have seen more of a list whenever I search
  for something, like a more extended version of the "Featured events" section.
  Almost like there should only be one of those sections and not both since they
  can show repeating information. (My preference is `Featured Events` for
  searching events and `Upcoming events` on the dashboard and homepage)
- I really like the approach to the profile and account. The QR code is a great
  idea for conferences and meetups. This combined with the privacy
  settings to be able to only show what I want is great!
